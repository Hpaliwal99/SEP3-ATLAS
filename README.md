**Group Apple Fritters**

Use java -jar SEP3-ATLAS.jar in the terminal in the extracted folder to run the program.

Use config.properties to edit various file path and constant variables.
The file paths have to be properly assigned before the config can work.

The code was run and tested on the database given in the paper, i.e. structured domains.txt (https://www.kaggle.com/datasets/mtatlas/atlas-analogy-structures, make sure to use absolute file path in config file), there is knowledge.txt provided in the package and can be used instead of the main database but is likely to not give enough results for coalesce Inferences.

It is recommended to run with high N for greedyMatching and getCandidateInferences for better results.

least effort way to run project is to clone GitHub repo (https://github.com/Hpaliwal99/SEP3-ATLAS.git).
